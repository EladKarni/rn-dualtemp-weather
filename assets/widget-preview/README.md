# Widget Preview Image

Create a screenshot of the weather widget for the Android widget picker.

Requirements:
- Size: 320x120 pixels (or larger, will be scaled)
- Format: PNG
- Name: weather.png
- Content: Preview of the text-based weather widget

The widget shows:
- Location name (top left)
- Last updated time (below location)
- Temperature (large, center-left)
- Weather description (below temperature)
- Humidity and wind speed (bottom)
- "Tap to open app" (bottom center)

Background: #1C1B4D (dark blue matching app theme)
Text: White/off-white colors

You can create this screenshot after building the app with widgets enabled.
